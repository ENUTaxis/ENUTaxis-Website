About Us

A simple online taxi booking system, allows users to book a taxi via a website. Would alert the user if no taxis/drivers were available for the time they require and would suggest the next available time for pickup. Each booking would be emailed to the bookings staff. E-mails would include client details as well as journey details and possibly allocate a driver. User would be provided with an estimated cost and the route mapped out using google maps.
